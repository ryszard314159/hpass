Edit Settings

  • Hint: leave it empty to change generic settings; Sometimes you may need to create hint/site specific settings, for example, if some site does not allow for special character in your generic settings, or demands longer/shorter password etc. In such a case, enter a hint, modify special character and/or length field
  • Secret (🤫): makes generated passwords unique
  • Special Character (🌶): one or more _.-!@$?% etc.
  • Length (📏): longer is better, note however that a lot of sites have unreasonably short upper limit - 15 seems to be a reasonable balance.
