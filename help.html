Installation and Settings

Press button to populate setting fields with default values
  • Secret (ü§´): random 8 character string, generated when app is installed, to make generated passwords unique for the user. You are welcome to change it if you wish.
  • Special character (üå∂): _ i.e. underscore
  • Length (üìè): 15
  • Burn (🔥): 0
Press button to save these values (except Burn) on your device. Password prompt (or hint) entered in the Enter Hint box at the top of the screen together with these settings determines the generated password i.e. with the same values always the same password will be generated.

NOTE: it is important to store Secret (and other values if different then default settings) in safe place in case you loose data on your device (or device itself). In such a case, you would need to enter these settings again in order to recreate your passwords - remember, these values (by design) are not stored in any central location, only on your device, and cannot be recovered if you loose them.

Usage

  1. Enter password hint in the Enter Hint box. You can use any string as a hint, but it is probably the easiest and most straighforward to use a secondary domain of the site for which you need a password e.g. abc for https://www.abc.com. For convenience you can simply copy and paste https://www.abc.com and abc will be automaticall extracted and used as a hint.
  2. You may want to enter custom value for Burn. This parameter is never stored, so in case your device is compromised and bad actors manage to steal your Secret this offers an extra security layer.
  3. Click button to generate password and copy to the clipboard.
  4. Paste generated password from the clipboard where you need it.
