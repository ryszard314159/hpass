Installation and Settings

Press Reset button to populate setting fields with default values:
  • Secret (ü§´): "Replace Me!" - you should change it to some value you are likely to remember.
  • Special Character (🌶): "_" i.e. underscore.
  • Length (üìè): 15
These three parameters are saved on your local device when you generate password.

Extra security

The purpose of two additional variables is to provide extra security layer in (very unlikely) case of your device being compromised and possibly your Secret, Special Character and Length parameters stolen. These variables:
  • Top Secret (🤐): any string you wish (default: empty string)
  • Burn (🔥): number, in 0-999999 range, of warm up rounds for Pseudo-Random Number Generator (PRNG) to use (default: 0)
are never stored on your local device or anywhere else, and discarted at the end of the session, or when the page is reloaded. After they are discarted they need to be entered again if you want to use them to generate password. This is a small price in convenience you may want to pay for high value passwords (bank accounts, social security, etc.) if you are concerned about possibility of data on your device being compromised.

Password prompt (or hint) entered in the Enter Hint box at the top of the screen, and Secret, Special Character Length, Top Secret are used to construct the seed for PRNG. This seed, together with Burn parameter (number of warm up rounds for PRNG), determines uniquely the generated password i.e. with the same values always the same password will be generated.

NOTE: it is important to store Secret (and other values if different then default settings) in safe place in case you loose data on your device (or device itself). In such a case, you would need to enter these settings again in order to recreate your passwords - remember, these values (by design) are not stored in any central location, only on your device, and cannot be recovered if you loose them - Top Secret and Burn are never stored.

Usage

  1. Enter password hint in the Enter Hint box. You can use any string as a hint, but it is probably the easiest and most straighforward to use a secondary domain of the site for which you need a password e.g. abc for https://www.abc.com. For convenience you can simply copy and paste https://www.abc.com and abc will be automaticall extracted and used as a hint.
  2. For extra security you may want to enter custom values for Top Secret and Burn. These parameters are never stored, so in case your device is compromised and bad actors manage to steal your data this offers an extra security layer.
  3. Click button to generate password and copy to the clipboard.
  4. Paste generated password from the clipboard where you need it.
