Start using HPASS:

  1. Enter Master Password (it is initially blank).
  2. Enter password hint in the Enter Hint box.
  3. Generate password by pressing Enter key or clicking on an the rigth arrow button in the top-upper corner. It will be automatically copied to the clipboard.
  4. Paste password from the clipboard where you need it.

Installation and Settings

When you first visit the HPASS site, generic settings are initialized to default values, encrypted (using Master Password as a key), and saved locally on your device.
  • Secret (ü§´): is initialized to a random string - something like "ypiGG8s1nUCNcv5l". Purpose of this is to generate passwords unique for each user, even if all other parameters are the same. You can keep it as is, or for enhanced security change it to a longer string. To do this you can use online random string generators (e.g. random.org/strings) or HASS itself. Alternatively, you can change it to some customized value you are likely to remember. Keep in mind, however, that - most likely - this would be less secure.
  • Special Character (üå∂): "_" i.e. underscore; multiple characters are allowed in this field.
  • Length (üìè): 15
Special Character and Length values are selected to satisfy the requirements for the majority of sites. You may need to change these values if some site enforces password requirements inconsistent with these values e.g. does not allow "_", requires password shorter that 15 charactes etc. These will be site-specific settings.

IMPORTANT: keep generic and site-specific settings stored in a safe location. You will need them to generate the same passwords in case data on your device are lost, or if device itself is lost.

To view, modify, save, and export / import all settings click gear button.

Multiple devices

You need to have the same settings on each device to generate the same passwords. To make this synchronization easier, settings can be exported () - encrypted or plain text - from one device and imported () on another using simple JSON file format.

Extra security

The purpose of two additional settings is to provide extra security layer in case your device is compromised and possibly your settings stolen.

These variables:

  • Top Secret (ü§ê): any string you wish (default: empty string)
  • Warm up (üî•): number, in 0-999999 range, of warm up rounds for Pseudo-Random Number Generator (PRNG) to use (default: 0)
are never stored anywhere, and discarted at the end of the session, or when the page is reloaded. They need to be entered every time you want to use them to generate password. This is a small price in convenience you may want to pay for high value passwords (bank accounts, social security, etc.) if you are concerned about possibility of data on your device being compromised.

Password prompt (or hint) entered in the Enter Hint box at the top of the screen, and Secret, Special Character, Length, Top Secret are used to construct the seed for PRNG. This seed, together with Warm up parameter (number of warm up rounds for PRNG), determines uniquely the generated password i.e. with the same values always the same password will be generated.
